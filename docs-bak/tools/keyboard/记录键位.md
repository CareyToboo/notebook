# 键盘键位

## 字母键

- qwert yuiop
- asdfg hjkl
- zxcvb nm
- 谐音法
  - 七碗鹅肉汤，已无一我婆
  - 爱上对方过后就哭了
  - 自行车未，帮你买

## 符号键

- 左边第一列
  - ~`
  - tab
  - caps-lock
  - shift
  - ctrl
- 右边小拇指
  - 第一行
    - 英文 `{}\`
      - 上档 `[]|`
    - 中文 `【】、`
      - 上档 `「」｜`
  - 第二行
    - 英文 `;'`
      - 上档 `:"`
    - 中文 `；‘`
      - 上档 `：”`
  - 第三行
    - 英文 `,./`
      - 上档 `<>?`
    - 中文 `，。`
      - 上档 `/《》？`
- 数字键
  - !
  - @
  - \#
  - $
  - %
  - ^
  - &
  - \*
  - (
  - )
  - \- =
  - _ +
