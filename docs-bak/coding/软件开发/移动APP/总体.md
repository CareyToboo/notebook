# 移动端APP

移动端APP整体分为2个方面，android/ios。其中有个各自的体系语言，从细节来说完全不相同。但是从整体概念来说又是1摸一样的。很多内容都和web类似，但是有海量的基础API东西要掌握，因此完全搞清楚也是很大鹅工作量，可能是费力不讨好的东西。因此需要趋利避害，去学习和研究核心的内容，而不要再细枝末节上耗费太多精力。大的方向来说，有很多相似性，因此掌握相似性来说很重要。
