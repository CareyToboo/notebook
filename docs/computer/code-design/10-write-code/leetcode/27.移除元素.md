# 27.移除元素

## 思路

- 遍历每一个元素
- 找出不需要删除的元素
- 设置到该数组对应位置
  - 需要定义 一个 记录下标值

## 伪代码

```js
var nums = [1,2]; target = 1; 
var i = 0;
// 遍历每一个元素
for (n of nums) {
  // 找出不需要删除的元素
  if (n != target) {
    // 设置到该数组对应位置
    nums[i] = n
    i += 1
  }
}

return  i
```
